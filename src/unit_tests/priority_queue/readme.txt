Проверка работоспособности очереди с кастомным компаратором(куча не на максимум, а на минимум)

Где:
1 - операция добавления числа
2 - операция удаления корня
3 - узнать макс(с нашим кооператором мин) число
4 - узнать размер


При запуске программы надо передавать 1 аргумент: название папки теста

Например: g++ -std=c++20 test.cpp && ./a.out test1
