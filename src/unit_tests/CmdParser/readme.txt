Структура: в тесте вводится командная строка, а в expected в правильном формате аргументы и строк, которые к ним относятся. Далее вводятся строки без аргументов(важно, что при вводе аргументов они должны быть отсортированы по алфавиту, а во всех остальных случаях по порядку ввода). Для лучше понимая лучше посмотреть сами тесты

Чтобы запустить тест1: g++ -std=c++20 test.cpp cmd_parser.cpp && ./a.out test1
